# 🤖 Auto Robô v2.0 - Análise Completa do Projeto

## 📋 Visão Geral

O **Auto Robô v2.0** é um sistema de automação web corporativa desenvolvido em **Node.js** com **Puppeteer**, criado por **Tiago Cesar Soares dos Santos (xAgron)** como parte dos estudos em **Análise e Desenvolvimento de Sistemas (ADS)**.

### 🎯 Objetivo Principal
Automatizar tarefas repetitivas em sistemas internos da Vivo/GVT:
- **GOI** - Gestão de Ordens de Instalação
- **WFM** - Workforce Management  
- **GPS** - Sistema de geoprocessamento

### 🔌 2. Atualizar o Servidor (`src/server/index.js`)

Altere apenas a parte do `socket.on('start-robot', ...)` para ler e injetar a lista de sistemas:

```javascript
socket.on('start-robot', async (data) => {
  if (robotProcess) {
    socket.emit('log', { level: 'warn', msg: '⚠ Robô já está rodando. Aguarde finalizar.' });
    return;
  }

  const { mode, systems } = data; // Ex: ['GOI', 'WFM']
  if (!systems || systems.length === 0) {
    socket.emit('log', { level: 'error', msg: '❌ Nenhum sistema selecionado.' });
    return;
  }

  socket.emit('log', { level: 'info', msg: `🚀 Iniciando Robô no modo: ${mode} | Sistemas: ${systems.join(', ')}` });

  // Diagnóstico pré-execução (mantido do seu código anterior)
  let canProceed = true;
  if (mode === 'DEV') {
    try {
      const res = await fetch('http://localhost:3456/login', { method: 'HEAD', signal: AbortSignal.timeout(3000) });
      if (!res.ok) throw new Error();
      socket.emit('log', { level: 'success', msg: '✅ Mock Server está ONLINE' });
    } catch {
      socket.emit('log', { level: 'error', msg: '❌ Mock Server NÃO está rodando! Execute: npm run mock' });
      canProceed = false;
    }
  } else {
    try {
      const dns = await import('dns').then(m => m.promises);
      await dns.lookup('goi.vivo.com.br');
      socket.emit('log', { level: 'success', msg: '✅ Rede corporativa acessível' });
    } catch {
      socket.emit('log', { level: 'error', msg: '❌ Sem acesso à rede corporativa. Modo PROD requer VPN/Cabo.' });
      canProceed = false;
    }
  }

  if (!canProceed) {
    socket.emit('status', { running: false });
    return;
  }

  // Injeta variáveis de ambiente
  const env = { ...process.env };
  env.TARGET_SYSTEMS = JSON.stringify(systems); // 👈 NOVA VARIÁVEL

  if (mode === 'DEV') {
    env.MODO_TESTE = 'true';
    env.LOGIN_URL = 'http://localhost:3456/login';
    env.AGENDAMENTO_URL = 'http://localhost:3456/agendamento/atendimento.php';
    env.WFM_LOGIN_URL = 'http://localhost:3456/login';
    env.WFM_URL = 'http://localhost:3456/agendamento/atendimento.php';
    env.GPS_URL = 'http://localhost:3456/login';
    env.HEADLESS = 'false';
  } else {
    env.MODO_TESTE = 'false';
  }

  const projectRoot = path.join(__dirname, '../..');
  const indexPath = path.join(projectRoot, 'index.js');
  if (!fs.existsSync(indexPath)) {
    socket.emit('log', { level: 'error', msg: `❌ Arquivo index.js não encontrado em ${indexPath}` });
    return;
  }

  robotProcess = spawn('node', ['index.js'], { cwd: projectRoot, env, stdio: ['ignore', 'pipe', 'pipe'] });
  socket.emit('status', { running: true });

  // ... (mantenha seu código de captura de stdout/stderr e robotProcess.on('close') exatamente como está) ...
});
```

### 🤖 3. Atualizar o Robô (`index.js`)

Substitua seu `index.js` por esta versão **modular e isolada por sistema**:

```javascript
// index.js - Entry Point com Execução Unitária por Sistema
import { initBrowser, closeBrowser } from './src/browser/index.js';
import { loginService, injectionService, navigationService } from './src/index.js';
import { logger } from './src/index.js';
import { captureAndSave } from './src/utils/screenshots.js';
import { sleep } from './src/utils/wait.js';

// Lê os sistemas selecionados pelo Dashboard (padrão: todos)
let TARGET_SYSTEMS = ['GOI', 'WFM', 'GPS'];
try {
  if (process.env.TARGET_SYSTEMS) {
    TARGET_SYSTEMS = JSON.parse(process.env.TARGET_SYSTEMS);
  }
} catch (e) {
  logger.warn('main', '⚠ Falha ao ler TARGET_SYSTEMS. Usando padrão: GOI,WFM,GPS');
}

console.log(`🎯 Sistemas selecionados: ${TARGET_SYSTEMS.join(', ')}`);

const runAutomation = async () => {
  let browser = null;
  let page = null;
  let globalError = false;

  try {
    logger.info('main', '📦 Inicializando navegador...');
    const { browser: br, page: pg } = await initBrowser();
    browser = br;
    page = pg;

    // === GOI ===
    if (TARGET_SYSTEMS.includes('GOI')) {
      try {
        logger.info('GOI', '🔐 Etapa: Login no GOI');
        await loginService.loginGOI(page);
        await sleep(1000);
        logger.success('GOI', '✅ Login GOI concluído');
      } catch (err) {
        logger.error('GOI', `❌ Falha no GOI: ${err.message}`);
        globalError = true;
      }
    }

    // === WFM ===
    if (TARGET_SYSTEMS.includes('WFM')) {
      try {
        logger.info('WFM', '🔐 Etapa: Login no WFM');
        await loginService.loginWFM(page);
        await sleep(1000);
        logger.success('WFM', '✅ Login WFM concluído');
      } catch (err) {
        logger.error('WFM', `❌ Falha no WFM: ${err.message}`);
        globalError = true;
      }
    }

    // === GPS ===
    if (TARGET_SYSTEMS.includes('GPS')) {
      try {
        logger.info('GPS', '🌐 Etapa: Abrindo GPS');
        await navigationService.openSystemTabs(browser, page); // Ajuste conforme sua implementação real
        await sleep(1000);
        logger.success('GPS', '✅ GPS carregado');
      } catch (err) {
        logger.error('GPS', `❌ Falha no GPS: ${err.message}`);
        globalError = true;
      }
    }

    // === Injeção D0 (só faz sentido se GOI estiver ativo) ===
    if (TARGET_SYSTEMS.includes('GOI') && !globalError) {
      try {
        logger.info('GOI', '💉 Etapa: Injeção D0');
        await injectionService.executarInjecaoD0(page);
        logger.success('GOI', '✅ Injeção D0 concluída');
      } catch (err) {
        logger.error('GOI', `❌ Falha na Injeção D0: ${err.message}`);
        globalError = true;
      }
    }

    // Screenshot final
    await captureAndSave(page, 'finalizacao');
    logger.success('main', '✨ Fluxo finalizado!');

  } catch (error) {
    logger.error('main', `💥 Erro crítico geral: ${error.message}`);
    if (page) await captureAndSave(page, 'erro_critico');
    process.exitCode = 1;
  } finally {
    if (browser) {
      logger.info('main', '🔒 Fechando navegador...');
      await closeBrowser(browser);
    }
    logger.info('main', '🔚 Processo encerrado.');
  }
};

// Executa sempre
runAutomation().catch(err => {
  console.error('💥 Erro fatal não tratado:', err);
  process.exit(1);
});
```

### 🔄 Como funciona agora?

1. Você abre o Dashboard em `http://localhost:3000`.
2. **Marca/Desmarca** os sistemas que quer executar (ex: só `GOI`).
3. Clica em `Modo Desenvolvimento` ou `Produção`.
4. O servidor injeta `TARGET_SYSTEMS=["GOI"]` no processo filho.
5. O `index.js` lê a variável e roda **apenas** os blocos `if` correspondentes.
6. Se um sistema falhar, os outros continuam rodando (isolamento de erros).
7. O Dashboard mostra logs prefixados por `[GOI]`, `[WFM]`, etc., facilitando a leitura.
